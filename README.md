# Hotel Management System

A simple console-based Hotel Management System built with Java using Object-Oriented Programming (OOP) principles and File I/O for data persistence.

## Features

- Search available rooms by category
- Book hotel rooms
- View all reservations
- Make payments for booked rooms
- Cancel reservations
- Save bookings and room availability to files
- Load saved bookings and room availability when the program starts
- Handles invalid user input using exception handling

## Technologies Used

- Java
- Object-Oriented Programming (OOP)
- ArrayLists
- File I/O (`PrintWriter`, `Scanner`)
- Exception Handling



## How It Works

1. The program loads previously saved room and reservation data.
2. A menu is displayed for the user.
3. Users can:
   - Search rooms
   - Book a room
   - View reservations
   - Make payments
   - Cancel reservations
4. When exiting, all bookings and room availability are saved automatically.
